# Mosul Maps

Playable Mosul map data belongs here.

Source sketches and reference plates should stay in `assets/mosul/source/` or `assets/mosul/references/`. Files here should be data the engine or tools can load.

The 2003 demo currently keeps visual floor PNGs in `assets/mosul/runtime/maps/market_commercial_streets_2003/levels/` and their LOS/movement metadata in `assets/mosul/manifests/market_commercial_streets_2003_building_levels.json`. Future collision masks, navigation grids, or tile products generated from that manifest can live here.
