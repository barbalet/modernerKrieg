# Mosul Runtime Assets

Generated runtime products belong here.

Files in this tree should be safe to rebuild from `assets/mosul/source/` plus metadata in `assets/mosul/manifests/`.

For the 2003 Market / Commercial Streets demo, runtime map levels are stored under `maps/market_commercial_streets_2003/levels/`. They are line-art PNG derivatives of the approved source stack and are paired with `assets/mosul/manifests/market_commercial_streets_2003_building_levels.json`.
